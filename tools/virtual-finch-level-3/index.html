Plan a path using direction and step count. The finish location changes each run.

Level 3

Plan your movement

The yellow target is randomized every run. Use the grid to count spaces, then choose a direction and step count to try to reach the target.

How to play

  • Look at the yellow target location
  • Use grid lines to count spaces
  • Select a direction
  • Enter a step count
  • Click Run and see if your plan works

Virtual Finch space

Watch the Finch move and bounce off walls. It stops when it reaches the target.

Start: 0, 0
Goal: Reach the yellow target

Ready for Level 4?

Level 4 adds multi-step programs or rule-based turning. Before moving on, make sure you can plan a route and explain why your plan worked or failed.

  • You can count grid spaces accurately
  • You can predict a bounce off a wall
  • You can adjust direction and steps based on where the target spawns
